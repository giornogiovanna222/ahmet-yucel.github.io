Bu alanda örnek icerik metinleri bulunmaktadır. Css özellikleriyle bu metinler arasinda boşluklar (line-height) ve hizalamalar (text-align) düzenlelleririz. Önce CSS ile kutulari / alaniari yan yana getirmeyi, ardından alt alta yerleştirmeyi öğrendik. Şimdi ise CSS seçicilerini kullanarak grup ve hiyerarşik seçimler yapmayı göreceğiz. Bu alan yan menü veya bilgi kismıdir. CSS ile genişlik (width), yükseklik (height), kenar boşlukları (margin/padding) ve kenarlik (border) özelliklerini kullanarak düzenleyebiliriz.
